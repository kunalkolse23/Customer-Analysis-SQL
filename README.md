

## 📊 Project Overview

This project focuses on analyzing customer purchasing behavior using SQL. The objective is to extract meaningful insights such as repeat customers, top buyers, and sales trends.

## 🛠️ Tools Used

* Oracle SQL
* SQL Developer

## 📁 Dataset

The dataset consists of two tables:

* Customers (customer_id, customer_name, city, signup_date)
* Orders (order_id, customer_id, order_date, total_amount)

## 🔍 Key Analysis Performed

* Identified repeat customers using GROUP BY and HAVING
* Analyzed total spending per customer
* Found top customers based on revenue
* Performed city-wise revenue analysis
* Analyzed monthly sales trends
* Used window functions (RANK) to find top customers

## 📈 Key Insights

* Repeat customers contribute significantly to total revenue
* Certain customers generate higher sales consistently
* Sales vary across different cities
* Monthly trends show changes in purchasing behavior

## 🚀 Conclusion

This project demonstrates how SQL can be used to analyze business data and generate actionable insights.
